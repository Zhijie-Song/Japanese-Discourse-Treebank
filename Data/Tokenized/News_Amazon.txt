Amazon 「 第 4 の 柱 」 に なる か 　 通信 事業 の 青写真 

 2023 年 11 月 13 日   2 : 00 

 米 アマゾン ・ ドット ・ コム が 通信 関連 事業 を 推し進め て いる 。 
 試験 段階 に 入っ た 衛星 インターネット に 加え 、 
 力 を 入れ て いる の が 、 
 通信 サービス の 末端 と なる 消費者 向け の スマート 機器 だ 。 
 それ ら の 機器 に 配信 する コンテンツ や 医療 ・ ヘルスケア サービス など と 絡め て 通信 事業 を 展開 すれ ば 、 
 定着 率 の 高い 大きな ビジネス に 育てる こと が できる 。 
 アマゾン が 通信 事業 に 関し て どんな 青写真 を 描い て いる の か 。 
 同社 の 直接 の 取り組み や 出資 、 
 買収 、 
 提携 し た 企業 から 探っ た 。 

 アマゾン が 有料会員 制度 「 プライム 」 、 
 クラウドコンピューティング 「 アマゾン ・ ウェブ ・ サービス （ AWS ） 」 、 
 オンライン 小売 事業 に 続く 「 第 4 の 柱 」 を 探し て いる の は 周知 の 事実 だ 。 
 通信 事業 へ の 取り組み が そう な の か も しれ ない 。 


 同社 が 通信 事業 を 拡大 すれ ば 、 
 配信 する コンテンツ や サービス だけ で は なく 、 
 配信 手段 自体 も 所有 できる よう に なる 。 
 プライム や スマート ホーム 製品 など 人気 の サービス と インターネット アクセス を セット で 販売 すれ ば 、 
 他社 の 単独 ブロードバンド ネットワーク より も 優位 に 立ち 、 
 既存 顧客 の 定着 率 を 高め られる 。 
 安定 し た ネット アクセス を 必要 と する 新規 顧客 も 獲得 できる だろう 。 


 さらに 、 
 顧客 から より 多く の データ を 収集 し 、 
 広告 や 小売 事業 と の 好 循環 を 生む よう に なる だろう 。 


 今回 の リポート で は CB インサイツ の データ を 活用 し 、 
 アマゾン の 2021 年 以降 の 買収 、 
 出資 、 
 提携 から 同社 の 通信 事業 で の 5 つ の 重要 戦略 を まとめ た 。 
 この 5 つ の 分野 で の アマゾン と の ビジネス 関係 に 基づき 、 
 各社 を 分類 し た 。 


 ・ 消費者 向け コネクテッド 製品 

 ・ 法人 向け コネクテッド 製品 

 ・ IoT コンポーネント （ 部品 ） 

 ・ 衛星 インターネット 

 ・ モバイル 通信 
 主な ポイント 
 ・ アマゾン が 通信 事業 の 5 つ の 重要 戦略 で 最も 積極的 に 推進 し て いる の は 消費者 向け コネクテッド 製品 （ スマート 機器 ） だ 。 
 消費者 の 日常生活 で さらに 大きな 役割 を 担おう と し て いる こと が うかがえる 。 
 モバイル 通信 や 衛星 インターネット で の 提携 は 、 
 自社 を 顧客 の 生活 の 中核 に 据え 、 
 「 必須 」 サービス を 提供 する こと に 注力 し て いる こと を 示し て いる 。 


 ・ 既存 プレーヤー と の 提携 は 通信 事業 の 拡大 を 支える だけ で なく 、 
 インフラ へ の 設備 投資 の 節約 に も なる 。 
 衛星 向け アンテナ の 有償 利用 サービス 「 AWS   Ground   Station 」 は 通信網 構築 の 起点 に なる が 、 
 アマゾン は 主に 既存 プロバイダー と の 提携 に より 事業 を 拡大 し て いる 。 
 と は いえ 、 
 衛星 インターネット サービス など の 事業 に なお 数十 億 ドル を 投じ て いる 。 


 ・ 法人 向け 事業 で は 、 
 垂直 統合型 IoT （ ネット 機器 ） サービス の 糸口 と し て 産業 分野 に 注目 し て いる 。 
 アマゾン は 高度 な 製造業 向け に コネクテッド 製品 を 手掛ける 企業 へ の 出資 や 提携 を 優先 し て いる 。 
 これ は いずれ 、 
 アマゾン の 物流 事業 に 相乗効果 を もたらす 可能性 が ある 。 


 消費者 向け コネクテッド 製品 
 アマゾン は 長年 、 
 アマゾン ブランド の スマート 機器 を 展開 し 、 
 人工知能 （ AI ） を 搭載 し た 音声 アシスタント 「 アレクサ 」 向け の 音声 テック 機能 に 資金 を 投じ て き た 。 
 アマゾン の アレクサシステム に 接続 する スマート 機器 は 4億 台 を 超える 。 


 こう し た スマート 製品 は 、 
 顧客 が コネクテッドホーム を 建てよう と する 際 に アマゾン の エコシステム （ 生態系 ） 内 に とどまる よう 促す 。 
 同社 が インターネット サービス も 提供 する よう に なれ ば 、 
 スマート 機器 と 優れ た ネット 接続 を セット 販売 できる よう に なる 。 


 アマゾン は 自前 の 機器 の 展開 や 22 年 の ロボット 掃除機 「 ルンバ 」 の 米 アイロボット 買収 に 加え 、 
 スマート サーモスタット の 独 タド （ Tado ） 、 
 支援 ロボット の 米 ラブラドール ・ システムズ （ Labrador   Systems ） 、 
 スマート コーヒーメーカー の 米 スピン （ Spinn ） など 多く の コネクテッド 製品 に 出資 し て いる 。 
 さらに 、 
 米 ゼネラル・エレクトリック （ GE ） や オランダ の フィリップス 、 
 ロボット 掃除機 の 北京 石頭 世紀 科技 （ ロボロック = 中国 ） など 消費者 向け スマート家電 を 手掛ける 大手 メーカー と 提携 し て いる 。 


 アマゾン は 医療 ・ ヘルスケア 戦略 も 推進 し て いる ため 、 
 医療 ・ ヘルスケア と 消費者 向け コネクテッド 製品 の 連携 も 増える だろう 。 
 例えば 、 
 アマゾン は 既に ヘルスケア 情報通信技術 の アスコム （ スイス ） や 米 セレニティー （ Serenity ） と 提携 し 、 
 自社 の スマート 製品 を 高齢者 の 生活 に もたらそう と 取り組ん で いる 。 


 法人 向け コネクテッド 製品 
 アマゾン の コネクテッド 体験 の 構想 は 家庭 だけ で なく 、 
 垂直 統合型 IoT サービス に も 及ぶ 。 
 画一的 な アプローチ で は なく 、 
 小売り から 自動車 まで 様々 な 部門 に 特化 し た 製品 や サービス を 手掛ける 可能性 が ある 。 


 アマゾン が 「 クライメート ・ プレッジ 」 や 「 アマゾン ・ インダストリアル ・ イノベーション ・ ファンド 」 など の コーポレート ベンチャーキャピタル （ CVC ） を 通じ て 気候 テック や 物流 に 力 を 入れ て いる こと を 考える と 、 
 産業 用 IoT （ IIoT ） を 起点 に する の は 自然 の 成り行き と いえる 。 


 例えば 、 
 インダストリアル ・ イノベーション ・ ファンド が 最近 出資 し た イスラエル の コアティゴ （ CoreTigo ） と 米 モジュール （ Modjoul ） の 2 社 は 、 
 いずれ も 製造業 の IoT に 力 を 入れ て いる 。 
 コアティゴ は 製造 機械 に 無線 機能 を 後付け する ハード と ソフト を 提供 し て おり 、 
 モジュール は 作業員 の ウエアラブル ベルト を 手掛ける 。 


 一方 、 
 AWS と 米 ベライゾン や 米 T モバイル US など 通信 キャリア と の 提携 は 、 
 高速 通信 規格 「 5 G 」 の エッジ コンピューティング の 向上 が 狙い だ 。 
 これ に より IoT 製品 の 遅延 が 減り 、 
 法人 向け アプリケーション を 増やせる 。 


 IoT コンポーネント （ 部品 ） 
 アマゾン が IoT コンポーネント を 手掛ける 企業 と の 提携 や 出資 を 進め て いる ため 、 
 外部 機器 メーカー は アマゾン の エコシステム に 参加 し やすく なっ て いる 。 


 例えば 、 
 アマゾン は 22 年 5 月 、 
 米 アンビエント ・ フォトニクス （ Ambient   Photonics ） の シリーズ A に リード 投資家 と し て 参加 し た 。 
 同社 は 室内 の 光 を 捉え て IoT 製品 に 電気 を 供給 する 太陽 電池 を 手掛ける 。 


 23 年 に は IoT 向け の アンテナ と モジュール を 生産 する 中国 の 移遠 通信（ケクテル） と 提携 し た 。 
 ケクテル の 部品 を 搭載 し た 機器 が 、 
 アマゾン 独自 の 通信 規格 「 アマゾン ・ サイドウォーク 」 に 自動 で 接続 できる よう に する 。 


 アマゾン ・ サイドウォーク は 無線 通信 や Wi-Fi の 帯域幅 を 必要 と し ない 機器 に 低 帯域幅 で 消費 電力 の 小さい ネットワーク を 提供 し 、 
 スマート 製品 の 接続 向上 を 目指す 。 
 この ネットワーク は 既に 米国 の 人口 の 90 % を カバー し て おり 、 
 通信 距離 は 最大 0.5 マイル （ 約 800 メートル ） だ と し て いる 。 


 23 年 9 月 に は 、 
 アマゾン ・ サイドウォーク 専用 の SoC （ システム ・ オン ・ チップ ） を 開発 する ため 、 
 米 半導体 メーカー の シリコン ・ ラボラトリーズ と の 提携 を 拡大 し た 。 


 衛星 インターネット 
 アマゾン は 低 軌道 （ LEO ） 衛星 を 低 遅延 、 
 大 容量 、 
 低コスト を 兼ね備え た インターネット 接続 の 未来 だ と 考え て いる 。 


 同社 は 通信 衛星 を 打ち上げ 、 
 早けれ ば 24 年後半 に も グローバル な 高速 インターネット 通信 を 提供 する 構想 「 プロジェクト ・ カイパー 」 を 掲げる 。 
 この プロジェクト で は 今後 数 年 で 少なく とも 100億 ドル （ 約 1兆5000億 円 ） を 投じ 、 
 3236 基 の 通信 衛星 群 を 配備 する 。 
 世界 の 3億 〜 4億 人 に サービス を 提供 できる 可能性 が ある 。 


 22 年 4 月 に は 米 ブルー オリジン 、 
 米 ユナイテッド ・ ローンチ ・ アライアンス （ ULA ） 、 
 仏 アリアン スペース と 提携 し 、 
 衛星 インターネット サービス を 実現 する ため に 過去 最大 規模 の 商業 衛星 打ち上げ を 発注 し た 。 
 23 年 10 月 に 初回 の 打ち上げ が 実施 さ れ た 。 


 アマゾン は 消費者 が 衛星 インターネット に 接続 できる 受信 端末 も 開発 し て いる 。 


 衛星 インターネット 市場 は 競争 が 激しく なり つつ ある 。 
 アマゾン は 異 業種 参入 組 と し て 、 
 衛星 インターネット サービス プロバイダー （ 衛星 ISP ） や 、 
 衛星 インターネット アクセス の 拡大 を 急ぐ 米 スペース X など と 争う こと に なる 。 
 例えば 、 
 スペース X の 「 スター リンク 」 は 既に 200万 人 以上 の 顧客 を 抱え て いる 。 


 モバイル 通信 
 アマゾン の 携帯 キャリア と の 提携 は 、 
 同社 が 仮想 移動体 通信 事業者 （ MVNO ） を 目指し て いる こと を 示し て いる 。 
 そう なれ ば 既存 キャリア の 回線 を 借り 、 
 自前 で インフラ を 構築 せ ず に 顧客 に 携帯 プラン を 提供 できる よう に なる 。 


 格安 携帯 サービス を 提供 する こと で さらに 多く の 顧客 を 獲得 し 、 
 プライム 会員 など サブスクリプション （ 定額 課金 ） の 付加価値 を 高め て 既存 顧客 を 囲い込む こと が できる 。 


 これ を 果たす ため 、 
 アマゾン は 23 年 に 米 新興 通信 事業者 ディッシュ ・ ネットワーク と 提携 し 、 
 プライム 会員 に データ を 無制限 に 使える 携帯 プラン 「 ブースト ・ インフィニット 」 を 提供 し た 。 
 23 年 9 月 に は 、 
 プロジェクト ・ カイパー の 始動 後 に これ を 活用 し て 欧州 と アフリカ の へき地 に ブロードバンド サービス を 拡大 する ため 、 
 英 ボーダフォン と 南アフリカ の ボーダコム と の 提携 を 発表 し た 。
