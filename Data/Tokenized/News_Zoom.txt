米 Zoom 、 
 出社 要請 の 舞台裏 　 リモート から ハイブリッド 

 日経 ビジネス 

 2023 年 11 月 17 日   2 : 00 

 ビデオ会議 システム 「 Zoom （ ズーム ） 」 の 運営 会社 が オフィス 回帰 を 始め た 。 
 夏 に メディア に 流出 し て 物議 を 醸し た 計画 通り 、 
 通勤圏 に 住む 従業員 に 週 2 回 の 出社 を 求め て いる 。 
 リモートワーク の 立役者 は なぜ 、 
 自己 否定 に なり かね ない 策 を 打ち出し た の か 。 
 幹部 が 舞台裏 を 明かし た 。 


 「 週 2 回 、 
 会社 に 行く こと に し た 」 。 
 10 月 上旬 、 
 米 ズーム ・ ビデオ ・ コミュニケーションズ の エリック ・ ユアン 最高経営責任者 （ CEO ） は シリコンバレー の 会議場 に 集まっ た 約 2600 人 の 顧客 を 前 に 、 
 そう 切り出し た 。 
 米紙 ニューヨーク・タイムズ （ NYT ） が 「 ズーム で さえ オフィス 回帰 」 と 報じ て 波紋 を 呼ん だ 施策 に つい て 、 
 CEO 自ら 言及 し た 格好 だ 。 


 ズーム の 方針 を おさらい し て おこう 。 
 同社 は 8 月 以降 順次 、 
 シリコン バレー の 本社 から 50 マイル （ 約 80 キロメートル ） 以内 に 住む 従業員 に 少なく とも 週 2 日 の 出勤 を 求め て いる 。 
 圏内 に は サンフランシスコ 市 など が 含ま れ 、 
 渋滞 が ひどく なけれ ば 1 時間 半 以内 に 移動 できる 距離 だ 。 
 技術 や 営業 と いっ た 部門ごと に 異なる 出社日 を 設け て いる 。 


 ズーム の 掲げ た 週 2 日 と いう 日数 その もの は 、 
 他 の テクノロジー 大手 と 比べ て 多く は ない 。 
 他 の 州 など から リモート で 働き 続ける 従業員 も いる 。 
 それ で も 、 
 新型コロナウイルス 禍 の 在宅勤務 を 支え て 一躍 有名 に なっ た ズーム が 「 出社 す べき 日 」 を 設け た こと は 物議 を 醸し た 。 
 米 ネット メディア の インサイダー は 「 リモートワーク 革命 の 死 」 と あおっ た 。 


 自己 否定 と も 捉え られ かね ない 方針 を 、 
 ズーム は なぜ 打ち出し た の か 。 
 製品 ・ 技術 部門 を 束ねる プレジデント の ベルチャミー ・ サンカルリンガム 氏 に 直接 疑問 を ぶつけ て みる こと に し た 。 
 同氏 は 2020 年 5 月 に ズーム に 加わり 、 
 コロナ禍 と その後 の 働き方 の 変遷 を 見つめ て き た 人物 だ 。 


 「 始まり は 半年 ほど 前 だっ た 」 。 
 サンカルリンガム 氏 は 開口 一番 、 
 こう 明かし た 。 
 NYT など の 記事 が 出 た 8 月 で は なく 、 
 3 月 まで 遡る 必要 が ある と いう 。 
 当時 の 米国 で は 会社 に 姿 を 見せ ない 従業員 に 業 を 煮やす 企業 が 相 次い で おり 、 
 米 スターバックス が 週 3 日 以上 は オフィス に 来る よう 求め たり 、 
 米 ウォルト ・ ディズニー が 月曜 から 木曜 まで を 「 対面 勤務 日 」 に 設定 し たり し て い た 。 


 ズーム で は 従業員 の 意思 に 任せ て い た が 、 
 製品 開発 を 率いる サンカルリンガム 氏 は 懸念 を 抱い て い た 。 
 「 顧客 と 同じ 環境 に 身 を 置か なけれ ば 、 
 良い 製品 は 作れ ない の で は ない か 」 。 
 出社 と リモート を 組み合わせる 「 ハイブリッド ワーク 」 を 支える 新製品 や 機能 の 開発 を 始め て い た もの の 、 
 本当 に 顧客 が 使い やすい もの を 作れ て いる の か 確信 を 持て なかっ た 。 


 そこ で まず 、 
 シリコン バレー の 本社 で 働く 230 人 の 製品 ・ 技術 チーム を 対象 に 「 水曜日 は 出社 しよう 」 と 決め た 。 
 会議室 に 設置 する ビデオ システム 「 ズーム ルーム 」 など の 自社 製品 を 使っ て 仕事 を 進め ながら 、 
 改善 点 を 洗い出す の が 狙い だっ た 。 
 「 最初 は 実験 と いう 位置づけ だっ た 」 （ サンカルリンガム 氏 ） 

 1 つ の 会議 で 4 室 を 専有 
 いざ 始める と 、 
 大小 様々 な 課題 が 見え て き た 。 
 例えば 、 
 オフィス に 来る 人 の ため に 開発 し た 会議室 の 予約 システム 。 
 ある 日 、 
 本社 に ある 4 つ の 会議室 が 予約 で 埋まっ て い た 。 
 調べる と 、 
 別々 の 会議室 に 入っ た 人 が 皆 、 
 同じ ズーム 会議 に 参加 し て い た 。 


 会議室 が 無尽蔵 に あれ ば 笑い話 で 済む が 、 
 オフィス の スペース は 限ら れる 。 
 別 の チーム の 打ち合わせ に 支障 を きたす 恐れ も ある 。 
 そこ で 「 同じ 会議 な の で 、 
 皆 で 一緒 に 会議室 を 使っ て み ては ？ 
 」 と 提案 する 機能 の 開発 に 着手 し た 。 
 サンカルリンガム 氏 は 「 オフィス に 行っ て い なけれ ば 、 
 肌 感覚 と し て 理解 でき なかっ た 」 と 振り返る 。 


 ハイブリッド ワーク に 身 を 投じ た 成果 は 、 
 目 に 見える 形 で 表れ つつ ある 。 
 10 月 上旬 に 開い た 顧客 向け の イベント で は 、 
 出社時 に 使う 席 の 予約 や 来訪者 を 管理 する システム を 発表 し た 。 
 会議室 から 複数人 で ビデオ会議 に 参加 し た 人 たち を 一人ひとり 分割 し て 見 やすく 表示 し 、 
 それぞれ に 名前 を 添える 新機能 など も 披露 し た 。 


 ユアン CEO は 「 ズーム を 使え ば すごく 効果的 な ハイブリッド ワーク が できる と いう 実例 に なり たい 」 と 話す 。 
 その ため に 、 
 8 月 から 出勤 を 求める 対象者 を 広げ 、 
 日数 も 週 2 日 に 増やし た 。 
 多く の 企業 が オフィス 回帰 の 理由 に 挙げる コミュニケーション の 活性化 だけ で なく 、 
 自社 製品 を 磨く と いう 明確 な 狙い が ズーム に は あっ た の だ 。 


 リモート の 主役 が ぶつかっ た 限界 
 リモートワーク の 主役 と いう 立ち 位置 から 脱皮 する 必要性 も 高まっ て い た 。 
 ズーム の 23 年 5 〜 7 月 期 の 売上高 は 11億3870万 ドル （ 約 1700億 円 ） で 、 
 前年 同期 から の 伸び率 は 4 % に とどまっ た 。 
 世界中 の 人 たち が 家 で 働い て い た コロナ禍 で は 同 4.7 倍 と いう 驚異的 な 伸び を 見せ た 時期 も あっ た が 、 
 22 年 半ば 以降 の 成長 曲線 は ほぼ 横ばい で 推移 し て いる 。 


 米国 の オフィス ワーカー を 対象 と する 米 ギャラップ の 継続 調査 に よる と 、 
 毎日 リモート で 働い て いる 人 の 割合 は 20 年 5 月 の 70 % から 23 年 5 月 に は 29 % まで 低下 し た 。 
 逆 に 増え た の が オフィス と 自宅 の 両方 で 働く 人 で 、 
 23 年 5 月 時点 で 全体 の 5 割 強 を 占め た 。 
 ハイブリッド ワーク を 追求 する 企業 を 取り込める か どう か は ズーム の 未来 に 直結 する 。 


 サンカルリンガム 氏 は 「 ハイブリッド ワーク は 複雑 だ 」 と 指摘 する 。 
 オンライン で の 使い 勝手 だけ を 追求 すれ ば 良かっ た 従来 と は 異なり 、 
 出社 する 人 に も 家 から 働く 人 に も 使い やすく なけれ ば いけ ない 。 
 日本 で は 対面 が 好ま れ やすい と いっ た 具合 に 、 
 地域 の 商習慣 や 業種 に よっ て も 傾向 は 異なる 。 
 一人ひとり の 働き方 と 業績 に 直結 する だけ に 、 
 企業 も 頻繁 に 制度 の 見直し を 行っ て いる 。 


 だ から こそ 、 
 米 マイクロソフト や 米 グーグル と いっ た 企業向け サービス の 大手 が 目配り し ない 「 かゆい ところ 」 に こだわっ て 、 
 活路 を 見いだそう と し て いる 。 
 通信 環境 が 悪い 場所 で の 途切れ にく さ を 追求 し て 頭 一 つ 抜け た ビデオ会議 システム で の 成功 パターン と 同じ シナリオ を 目指す と いう わけ だ 。 


 オフィス 回帰 を 選ん だ ズーム は ハイブリッド の 世界 で 再び 輝け る の か 。 
 その 成否 は 、 
 人々 の 働き方 に も 影響 を 及ぼす 。
