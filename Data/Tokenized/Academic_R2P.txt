「 保護 する 責任 」 を 通し て みる 「 国際 秩序 論 」 — — 国際連合 事務局 の 権力 性 
 赤星   聖 
 2021   年 2   月 ， ミャンマー 国軍 に よる クーデター ， 同年 8   月 ， アフガニスタン に おけ る タリバン に よる 政権 奪取 ， 2022   年 2   月 ， ロシア に よる ウクライナ 侵略 と ， 過去 2   年間 ， 国際的 に 衝撃 を 与える 事件 が 続い た 。 
 とりわけ ， ロシア に よる ウクライナ 侵略 は ， 第 二 次 大戦 後 に 確立 し た 国際 秩序 の 動揺 を 招い て いる と 指摘 さ れる が ， ここ で いう 「 国際 秩序 の 動揺 」 と は ， ロシア に よる 違法 かつ 一方的 な 武力 行使 に よる ウクライナ 国家 の 主権 侵害 を 指し て いる と 考え られる 。 
 本書 が 分析 する 「 介入 と 国家 主権 に 関する 国際 委員会 （ ICISS ） 」 に よっ て 提示 さ れ た 「 保護 する 責任 （ R 2 P ） 」 概念 も また ， 国家 主権 と 武力 不 行使 原則 と いう 戦後 国際 秩序 の 根幹 を 対象 と し た もの で あっ た 。 

 本書 は ， 1990   年代 から 2015   年 に かけ て   R 2 P   概念 の 歴史的 変遷 を 国連 文書 や インタビュー を 用い て 丹念 に たどる こと を 通し て ， 「 『 望ましい 国際 秩序 』 を めぐる 議論 （ 国際 秩序 論 ） 」 （ 7   頁 ） が 変遷 し て き た と 論じる 。 
 R 2 P 概念 が 国際 秩序 論 を 考察 する うえ で 適切 な 理由 は ， 「 人道的 介入 」 の 是非 に 対する ひと つ の 回答 と し て 提示 さ れ た   R 2 P   概念 が ， 二 度 の 大戦 を 経 て 確立 し て き た 内政 不干渉 原則 および 武力 不 行使 原則 と いう 国際 秩序 の 最深 層 の 修正 を 要求 する もの で あっ た ため で ある 。 
 以下 本書 の 内容 を 概観 し たい 。 

 序論 で は ， 本書 の 目的 および   R 2 P   概念 を 分析 する 理由 が 述べ られる と とも に ， 自ら の 立ち位置 ・ 貢献 を 明確化 する ため ， 「 国際 政治 思想 研究 」 全体 を とらえる 見取り図 が 提示 さ れ て いる 。 
 この 見取り図 は ， 専門 で は ない 評者 に とっ て 全体像 を 把握 する に 資する 有用 な 整理 で あっ た 。 
 続い て 第 1   章 は ， 先行 研究 を 批判 し た うえ で ， 本書 が 依拠 する 分析 枠組み ・ 概念 セット が 示さ れる 。 
 先行 研究 の 多く は   R 2 P   概念 を 一面的 に 理解 し ， 同 概念 の 変遷 を 論じ た 数 少ない 研究 も ， R 2 P   概念 を 「 規範 」 と とらえ ， その 規範 自体 の 変化 を 分析 の 焦点 と し た ため ， 議論 の 射程 に 国際 秩序 論 が 入ら ない と する 。 
 そこ で 本書 は ， ①三 つ の 時期 （ R 2 P   概念 の 萌芽 期 ， 論争 期 ， 推進 期 ） ， ②三 つ の 国連 （ 加盟国 ， 事務局 ， 市民 社会 ） ， ③三 つ の 系譜 （ 正戦 ／ 介入 論 ， ガバナンス 論 ， 紛争 予防 論 ） と いう 分析 枠組み を 提示 し た 。 
 ①② も 重要 で ある が ， 本書 の 最大 の 貢献 は ， ③ を 提示 し た こと で あり ， さらに は ガバナンス 論 と 紛争 予防 論 を 明示 的 に   R 2 P   概念 研究 に 導入 し た こと で あろう 。 

 第 2 章 で は ， ICISS   報告書 に 至る 歴史的 背景 を 踏まえ た うえ で ， R 2 P   概念 に 含ま れる 三 つ の 系譜 が 説明 さ れる 。 
 同 報告書 は ， 法 の 支配 や 人権 の 国際的 保障 を 重視 する 国際 立憲 主義 に 基づき ， 市民 を 保護 する 責任 を 果たさ ない 政府 に 対し て 国際 社会 に よる 武力 介入 も 可能 で ある と する 正戦 ／ 介入 論 が 中心 で あっ た 。 
 対し て ， 第 3   章 で 議論 さ れ た の は 2005   年 世界 サミット に 至る 過程 で あり ， この 中 で 正戦 ／ 介入 論 が 後景 に 下がり ， 国家 主権 を 中心 と し 各国 の グッド ・ ガバナンス を 補完 する こと を 国際 支援 の 目的 と する ガバナンス 論 を 中心 と し た   R 2 P   概念 の 理解 が 国家間 で 合意 さ れ た 。 
 この 背景 と し て ， アメリカ の 単独 行動 主義 と   R 2 P   概念 が 結び付け られ ， 武力 介入 を 中心 と する   R 2 P   概念 に 同意 が 得 難く なっ た こと ， 途上国 グループ が 開発 支援 を 強調 し た こと など を 指摘 し て いる 。 
 第 4   章 で は ， 2009   年 国連 事務 総長 報告書 を 中心 と し て ， 紛争 予防 論 の 系譜 に 連なる 国連 事務局 に よる 早期 警報 体制 が 強化 さ れる 過程 が 追跡 さ れる 。 
 2011   年 リビア 介入 前後 に ， 多国間 議論 の 場 に おい て   R 2 P   概念 の 否定的 な 理解 あるいは その 濫用 が 認め られ た こと で ， 学術的 に も   R 2 P   概念 の 評価 が 揺らぎ を 見せる が ， 本書 は この 時期 を 含め て   R 2 P   概念 の 「 推進 期 」 と みなし て いる 点 が 特徴 で あろう 。 

 第 5   章 および 終章 は ， これ ら の 事例 研究 を 踏まえ た 権力 に 関する 議論 で あり ， 本書 に おけ る ハイライト で も あろう 。 
 ICISS   報告書 が 提示 し た R 2 P   概念 は ， 人道 危機 対応 の 権限 を 国連 安全保障理事会 に 集約 し ， 直接 介入 を 進展 さ せる 国際 立憲 主義 に 基づく もの で あり ， カール ・ シュミット の いう 「 緊急権 と し て の 超 国家 権力 」 を 国際 秩序 に 導入 し た もの で あっ た 。 
 しかし ， 世界 サミット に 至る 議論 の 中 で ， 各国 の 国内 統治 を 重視 し ， 国際 社会 は それ を 支援 する と いう ミシェル ・ フーコー の いう 「 統治 権力 と し て の 超 国家 権力 」 が 前面 に 顕れ た 。 
 さらに ， 2009   年 事務 総長 報告書 を 契機 に 進め られ た ， 国連 事務局 を 中心 と し た 早期 警報 能力 の 強化 と は ， 紛争 を 予防 する 「 グローバル な 監視 体制 」 の 構築 で あり ， 「 規律 権力 と し て の 超 国家 権力 」 （ フーコー ） を 通し て ， 国際 秩序 論 が 国際 機能主義 へ と 変遷 し つつ ある こと が 仮説 的 に 論じ られ ， この 動き は 立憲 主義 的 な 正当性 の 観点 から 問題 も 多い こと を 指摘 し て いる 。
